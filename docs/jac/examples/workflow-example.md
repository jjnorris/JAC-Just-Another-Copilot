# Workflow example

Example session using JAC manually:

1. Open `instructions.md`.
2. Run `workflows/intake-and-clarify.md` to classify ambiguity.
3. Fill `templates/correctness-contract.json`.
4. Run `workflows/plan-review.md` and choose `always_proceed` or `request_review`.
5. Implement one bounded slice with `workflows/bounded-implementation.md`.
6. Finish with `workflows/verify-before-claim.md` and `templates/verification-report.json`.
