# Workflow example

Example session using JACK manually:

1. Open `instructions.md`.
2. Run `docs/jacks/workflows/intake-and-clarify.md` to classify ambiguity.
3. Fill `docs/jacks/templates/correctness-contract.json`.
4. Run `docs/jacks/workflows/plan-review.md` and choose `always_proceed` or `request_review`.
5. Implement one bounded slice with `docs/jacks/workflows/bounded-implementation.md`.
6. Finish with `docs/jacks/workflows/verify-before-claim.md` and `docs/jacks/templates/verification-report.json`.
