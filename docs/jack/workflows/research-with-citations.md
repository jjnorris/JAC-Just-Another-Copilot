# Research with citations

## goal
Produce current or externally grounded recommendations without collapsing fact and inference.

## when to use
- Time-sensitive questions.
- Unverified platform behavior.
- Security, compliance, or license questions.

## prerequisites
- A clear research question.
- Criteria for what decision the research informs.

## steps
1. State what is unknown.
2. Gather primary sources first.
3. Grade each source.
4. Separate facts, inferences, and recommendations.
5. Cite sources in the recommendation.
6. Mark any untrusted content as untrusted.

## pause points
- When only weak sources are available.
- When research findings change risk class.

## evidence to collect
- citation list
- trust grades
- recommendation rationale

## stop conditions
- Evidence base is too weak for a confident recommendation.

## escalation or review path
- Escalate if the decision is high-risk and evidence remains mixed.

## final report contract
Report verified facts, inferences, recommendation, source grades, and next validation step.
