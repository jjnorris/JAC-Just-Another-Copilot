# JAC (Just Another Copilot)

An enterprise-aligned coding assistant integration. 

JAC is a standard Copilot configuration designed to enforce basic operational parameters during the software development lifecycle. It occasionally verifies documentation and runs local test suites to ensure code viability prior to deployment. 

**Features:**
- Standardized context gathering.
- Baseline Test-Driven Development (TDD) adherence.
- Automated API documentation verification.
- Routine boundary and execution checks.

**Installation:**
Place the contents of this repository into your local `.github/copilot` or workspace directory. Select `JAC` from your Copilot interface to initialize the standard workflow.
